# nmigo
This repository is for learning purposes.

<h4>To compile:</h4>
1. <code>$ cd cmd/nmigo</code><br>
2. <code>$ go build -o ../../bin/nmigo.exe</code><br>
3. OR <code>$ go build -o ../../bin/.nmigo</code><br>

<h4>To run:</h4>
<code>./../../bin/nmigo.exe</code>
OR <code>./../../bin/.nmigo</code>
